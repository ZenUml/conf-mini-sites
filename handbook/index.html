Overview
● BUILD GATED

Confluence × Cloudflare Workers for Platforms Host a multi-file static bundle, live in Confluence.

Conf Mini-Sites lets a user upload an AI-generated interactive mini-site — a clickable prototype, a filterable dashboard, a troubleshooting tool — and embed it as a live object on a Confluence page. This handbook is the design of record: architecture, threat model, domain model, flows, and the path to build.

● Build gated — design only Cloudflare WfP Atlassian Connect Auth gateway = CVSS-9.1 class $25/mo to host · ~$500–700 to run safely
⚑
Status: do not build yet. Blocked on four gates — security/residency sign-off, demand-to-pay (≥3 beyond the anchor team), signed acceptance criteria, and EV-vs-next-feature. See Gates & Positioning. Everything here is design and is fine to produce now.

Architecture at a glance

One dispatch namespace; one non-routable user Worker per macro instance serving its bundle via Workers Static Assets. The dispatch Worker is the auth gateway — the single network entry point and the only thing standing between a viewer and the bundle bytes.
